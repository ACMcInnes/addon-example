Test integration for the [Neto eCommerce](https://netohq.com/) Platform, connects securely via OAuth, no API Keys required! Stay tuned for updates

## Commands

| Command                   | Action                                               |
| :------------------------ | :--------------------------------------------------- |
| `pnpm install`            | Installs dependencies                                |
| `pnpm dev`                | Starts local dev server at `localhost:3000`          |
| `pnpm preview`            | Preview production build                             |
| `pnpm build`              | Create a production build                            |
| `pnpm version <version>`  | Bump project version, e.g. `major`, `minor`, `patch` |
| `vercel`                  | Run CLI commands for Vercel, e.g `vercel env pull`   |
